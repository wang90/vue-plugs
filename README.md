# vue-plugs
# 这是一个基于vue组建应用的类型库
基础库vue >= 1.0  

#### 监听div滚动  v-load
1.引用  v-load.js    
2.绑定js    
`````
<div v-load={"top":scrollTop,"bottom":scrollBottom}"></div>
`````
3.参数
``````
@parms scrollTop
@parms scrollBottom 
``````

#### 计算时间戳转换 v-time
1.引用  v-item.js    
2.绑定js    
`````
<div v-time="time"></div>
`````
